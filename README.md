# 个人脚手架工具

## 安装
```
npm install zero-personal-cli -g
// 或
yarn global add zero-personal-cli
```

## 使用
```
// 初始化微前端项目
zero init micro [project-name]

// 初始化PC端项目
zero init vue [project-name]

// 初始化H5项目
zero init vue-h5 [project-name]

// 查看所有模板及模板地址
zero list
```

## 模板列表
- micro qiankun微前端框架
- vue 基于vue的PC端框架
- vue-h5 基于vue的移动端框架
